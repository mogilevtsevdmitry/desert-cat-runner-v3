# Desert Cat Runner v3

Desert Cat Runner — это динамичная браузерная игра, разработанная с использованием современных веб-технологий. Проект сфокусирован на высокой производительности, чистой архитектуре и качественном пользовательском опыте.

## 🚀 Tech Stack

- **Framework**: React 18+
- **Language**: TypeScript (Strict Mode)
- **Build Tool**: Vite
- **State Management**: Zustand
- **Styling**: Tailwind CSS
- **Deployment**: Vercel / GitHub Pages
